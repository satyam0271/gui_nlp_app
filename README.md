# 🧠 NLP Desktop Application (Python + Tkinter)

A GUI-based NLP Desktop Application built using Python and Tkinter that integrates the NLP Cloud API to perform:

- ✅ Sentiment Analysis  
- 📰 Headline Generation  
- 🏷 Named Entity Recognition (NER)  

The application also includes a complete user authentication system (Registration & Login) with JSON-based data storage.

---

## 🚀 Features

### 🔐 User Authentication
- User Registration
- Secure Login with Email & Password
- User data stored locally in JSON format using Python dictionary

### 🤖 NLP Functionalities (via NLP Cloud API)
- Sentiment Analysis
- Headline Generation
- Named Entity Recognition (Entity Extraction)

### 🖥 GUI Interface
- Built using Tkinter
- Simple, clean, and interactive user interface
- Desktop-based application

---

## 🛠 Tech Stack

- **Python**
- **Tkinter** (GUI Framework)
- **NLP Cloud API**
- **JSON** (for local user data storage)

---


---

## ⚙ How It Works

1. User registers with email & password
2. Credentials are saved in `users.json`
3. User logs in
4. After authentication, user can:
   - Analyze sentiment of input text
   - Generate headlines
   - Extract named entities
